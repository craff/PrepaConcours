Compilation du cours

Si vous voulez contribuer à ce cours, n’hésitez pas à sousmettre des «issue» sur github, voire des «pull-request».

Pour compiler le cours (c’est à dire produire la version html) il vous faut:

  • bash
  • pandoc
  • lua (pour un script pandoc)
  • une distribution récente de LaTeX incluant tikz

Si vous avez tout ça, pour compiler il suffit d’exécuter ./build.sh à la racine de ce dépot.

Pour faire le ménage ./clean.sh
